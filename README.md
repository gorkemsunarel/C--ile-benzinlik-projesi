# Gas Station Simulation (C#)

## Project Overview
This is a C# gas station simulation application that demonstrates a simplified fueling process.  
The program simulates cars approaching a gas station with different fuel types (Gasoline, Diesel, LPG).  
When a user clicks on a car, the refueling process starts. Once the car is fully fueled, its status turns green and it drives away.

## Features
- Simulation of vehicles arriving at a gas station
- User interaction to trigger refueling
- Supports multiple fuel types (Gasoline, Diesel, LPG)
- Visual status change when fueling is completed (green indicator)

## Technologies Used

- **.NET (Console or Windows UI depending on implementation)**
- Basic Object-Oriented Programming (OOP)
